# 目录结构

```
|-- build                           // 项目构建(webpack)相关代码
|-- config                          // 项目开发环境配置
|-- dist                            // 运行npm run build 生成的生产环境代码目录
|-- core                            // 公共内容（包含api、组件、服务等）
|-- |-- bridge                      // jsapi
|-- |-- debug                       // vconsole 调试配置
|-- |-- helper                      // core目录内依赖的服务api
|-- |-- init                        // 页面初始化处理
|-- |-- lib                         // 项目库文件
|-- |-- public                      // 组件目录
|-- |-- |-- assets                  // 图片资源目录（组件依赖资源）
|-- |-- |-- components              // 组件
|-- |-- |-- |-- base                // 基础组件
|-- |-- |-- |-- libs                // vux的libs
|-- |-- |-- |-- project             // 项目组件
|-- |-- |-- |-- styles              // vux样式
|-- |-- |-- |-- vux-support         // vux的一些函数
|-- |-- |-- less                    // 公共样式目录
|-- |-- utils                       // 服务api
|-- |-- index.js                    // core对外暴露的api
|-- doc                             // 项目相关文档目录
|-- mock                            // 开发测试数据目录
|-- node_modules					          // 项目依赖包
|-- src                             // 源码目录
|   |-- assets                      // 静态资源目录(每业务包一个目录，存储图片资源)
|   |-- modules                     // 功能模块目录
|   |   |-- demo                    // 离线包模块一级目录（包名）
|   |   |-- assets                  // 离线包静态资源文件目录
|   |   |   |-- demo                // 离线包模块二级目录（模块名）
|   |   |   |   |-- index           // 离线包模块二级目录下index.html源代码目录（页面名称）
|   |   |   |   |   |-- components  // 页面自定义组件目录
|   |   |   |   |   |-- doc         // 页面相关文档目录
|   |   |   |   |   |-- utils       // 页面相关工具类目录
|   |   |   |   |   |-- index.scss  // 页面scss文件
|   |   |   |   |   |-- index.vue   // 页面vue文件
|   |   |   |   |   |-- main.js     // 页面主入口文件
|   |   |   |-- conf.json           // 离线包模块配置文件（包含：每个页面配置。标题、启动参数（导航栏背景颜色、标题颜色等））
|   |-- main.js                     // 全局引用文件(页面主依赖文件)
|-- static                          // 静态资源目录（全局资源包的内容目录）
|-- .babelrc                        // Babel配置文件
|-- .gitignore                      // git忽略策略配置文件
|-- .postcssrc.js                   // postcss配置文件
|-- package.json                    // 项目依赖配置文件
|-- template.html                   // 项目html通用模板
```
