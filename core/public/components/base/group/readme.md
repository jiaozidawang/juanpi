



#### 属性

名字	| 类型	| 默认值	| 说明
-----|------|------|------|-----
title	| string	|	| 分组标题
title-color	| string	| | 分组标题文字颜色
label-width	| string	|	| 为子元素设定统一label宽度
label-align	| string	|	| 为子元素设定统一对齐方式
label-margin-right	| string	|	| 为子元素设定统一的右边margin
gutter	| string		| | 设定group的上边距，只能用于没有标题时
hasBg | Boolean |  | 设置是否有背景


#### 插槽
名字	| 说明
----|----
默认插槽	| 子组件插槽
title	| 标题插槽
